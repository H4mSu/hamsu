# 깃 허브 업로드 순서
1. 수정후 저장 M 생성 
2. add를 사용하자
    ```
    git add . -> 모두 업로드
    git add 파일명 -> 해당 파일만 업로드
    ```
3.commit 사용
```
git commit -m "깃에 올라오는 버젼 설정" 
git commit - 현재 상태 저장
```
4.로그 한번 확인
```
git log
```
5.push로 허브에 올리기
